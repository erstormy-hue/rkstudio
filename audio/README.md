# Audio teaser structure

This folder stores teaser snippets only (short MP3 previews), never full songs.

Genre folders are aligned with the website menu:

- `film-music/`
- `podcast-intro/`
- `pop/`
- `rnb/`
- `edm/`
- `classical-ambient/`
- `country-americana/`
- `alternative/`
- `jazzy/`

Current teaser placement rule:

- Place all R&B teaser MP3 files in `rnb/`.
- Keep future teaser uploads in their matching genre folder.

Empty folders include a `.gitkeep` placeholder so Git keeps the structure.
